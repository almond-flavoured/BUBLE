BUBLE(Buy Ur Best Limited Edition)은 감당 가능한 신발 한정판을 추구하는 플랫폼 서비스로

제테크용 이외에 신을 수 있는 한정판 중고거래를 중개합니다

✅ 신발의 사용 정도에 따른 등급제를 통해 보다 부담 없는 가격을 제공합니다

✅ 구매, 판매에서 그치지 않고 월별 대여 시스템을 제공합니다

✅ 시세 그래프를 제공합니다
